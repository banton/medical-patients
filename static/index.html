Military Patient Generator

Simple Interface for Patient Data Generation

Configuration Files

The patient generator uses the following JSON configuration files located in the patient_generator/ directory:

demographics.json - Contains nationality data, names, and ID formats for NATO nations
fronts_config.json - Defines battle fronts with casualty ratios and nationality distributions
injuries.json - Specifies the distribution of injuries (disease, non-battle injuries, battle injuries)

To modify the generation parameters, edit these JSON files directly before clicking Generate.
